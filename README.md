# Aravind Jegadesan Portfolio

Professional static portfolio site for Aravind Jegadesan, Senior Software Engineer - Backend.

## Local Preview

Open `index.html` in a browser, or serve the folder with any static server.

## Verification

```bash
node tests/verify-site.mjs
```

## GitHub Pages

This site is ready to host from the repository root:

1. Push this repository to GitHub.
2. Open repository settings.
3. Go to Pages.
4. Set source to the default branch and folder to `/root`.
5. Save and wait for GitHub Pages to publish the site.
